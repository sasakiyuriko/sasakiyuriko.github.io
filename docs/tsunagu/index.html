クラウドソーシングの利用の際、必要となる仲介手数料。つなぐでは他社を圧倒する低手数料にすることで利用者の負担を軽減し、誰もが気軽に利用できる環境を提供します。
手数料は販売価格の5.5％（税込）であり、クリエイター(販売者)の報酬から差し引かれます。 クライアント(依頼者)は手数料不要となります。

個人間で売買を行うと、支払い方法が制限され、特に購入者（クライアント）側は不便に感じます。全ての人がストレスなく購入できるように、クレジットカード払い・銀行振込・コンビニ決済に対応しています。
※現在のところ、クレジットカード、デビットカード、国際カードブランド付き、プリペイドカードの利用が可能。その他の決済方法に関しては順次対応予定。

初心者クリエイターが実績をつくれるコンテンツやＳＮＳによる集客のお手伝いをすることで、クリエイターとして活動することに不安を感じる方のサポートを行います。
